This directory is managed by scripts/update_org_branch_health.py.

Contents are regenerated automatically by the "Update org branch health" workflow
and committed back to this repository.

Files written here:
  - <repo>-dev.svg   one SVG badge per repo for the dev branch
  - <repo>-main.svg  one SVG badge per repo for the main branch
  - status.json      machine-readable snapshot of the last scan
